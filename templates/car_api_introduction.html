Example of Basic Django REST Framework - Car API Project

Check basic car makes and models existence in database.
Add, rate, delete and view most rated cars

API is interacting with an external API - https://vpic.nhtsa.dot.gov/api/ - for car existence validation.

You can interact with the API only programmatically. This means you have to use tools like POSTMAN, or send requests programmatically to the relative endpoints. Interracting with API does not require authentication

Find more information about the project on GITHUB: https://github.com/klimuntowskirafal/car_api

Endpoints:
more detailed description to the endpoints with described data structure for POST/DELETE requests can be found in 'endpoints.txt'

  • /cars/

    send GET request

  • /cars/

    send POST request

  • /cars/{ id }/

    send DELETE request with provided ID

  • /rate/

    send POST request

  • /popular/

    send GET request
